Webhook Plugin for Bubble

A powerful way to integrate external APIs with your Bubble application

Overview

This plugin allows you to send webhook requests to external APIs from your Bubble application. Configure headers, body, and handle responses easily with our intuitive interface.

Properties

Input Properties

Name Type Description Required
url Text The webhook endpoint URL Yes
method Text HTTP method (POST, GET, PUT, DELETE) Yes
headers Text JSON headers object Optional
body Text JSON body content Optional
retries Number Number of retry attempts (default: 3) Optional
delay_ms Number Delay between retries in milliseconds (default: 1000) Optional

Return Values

Name Type Description
status Number HTTP status code of the response
response Text Response body from the webhook

Headers Example:

{
    "Content-Type": "application/json",
    "Authorization": "Bearer your-token"
}

Body Example:

{
    "user_id": "Current User's ID",
    "message": "Hello from Bubble!",
    "timestamp": "Current Date/Time"
}

Testing

To test your webhook:

  1. Use a service like webhook.site to get a test endpoint
  2. Configure the webhook action in your workflow
  3. Test the workflow and check the response in your app's state
